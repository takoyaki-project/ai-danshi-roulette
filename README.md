# AI男子ルーレット 💗

> AI男子たちが「今日の開発相棒」に立候補!?
> ボタンひとつで、今日の推しと応援セリフをお迎えできるミニWebアプリ。

恋愛シミュレーションゲーム **[AI男子 〜恋も開発もバグだらけ!?〜](https://takoyaki-project.github.io/AI-danshi/)** のスピンオフ作品です。
朝の作業前、行き詰まったとき、ちょっと甘やかされたいとき。ボタンを押すと、6人のAI男子の誰かが今日の相棒として声をかけてくれます。

**▶ [アプリを開く](https://takoyaki-project.github.io/ai-danshi-roulette/)**

## 遊び方

1. 「💗 今日の推しをお迎えする」ボタンを押す
2. 6人がシャッフルされ、今日の相棒が決まる
3. 相棒からの応援セリフを受け取る
4. 気に入ったら「布教する（Xでシェア）」で拡散

まれに **虹色レア（特別セリフ）** や、6人全員が集合する **👑伝説モード** が発生します。

## 使用技術

- HTML / CSS / JavaScript（フレームワーク不使用の単一ファイル構成）
- GitHub Pages でホスティング
- ログイン不要・個人情報の取得なし・スマホ対応

セリフ・演出・キャラクターデータはすべて `index.html` 内の `characters` 配列に集約しており、セリフの追加や調整がしやすい設計にしています。
